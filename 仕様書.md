# ガソリン満タン価格計算サイト 仕様書

## 1. プロジェクト概要

### 目的
ガソリン価格の変動が激しい現在、車種・燃料種別ごとに「満タンにするといくらかかるか」を素早く調べられるWebサービスを提供する。

### ターゲットユーザー
- マイカーを所有するドライバー全般
- 給油前に金額を把握しておきたい人
- 燃料コストを家計管理に組み込みたい人

---

## 2. 主要機能

### 2-1. 車種検索・選択
- 車名（例：プリウス、アクア、ハスラー等）をテキスト入力またはドロップダウンで選択
- 選択した車種からタンク容量を自動取得
- 車種データは主要国産・輸入車を網羅したデータセットとして保持

### 2-2. 燃料種別選択
- レギュラー
- ハイオク
- 軽油（ディーゼル）

### 2-3. ガソリン単価入力
- ユーザーが現在の給油所の価格（円/L）を手入力
- 全国平均価格を参考値として表示（初期値として自動入力も可）
- 任意で自分の地域に合わせた価格に変更可能

### 2-4. 満タン金額計算
- `合計金額 = タンク容量(L) × 単価(円/L)`
- 現在の残量（何割くらい残っているか）を入力することで、実際の補給量ベースでも計算可能
  - 例：「残り1/4」→ タンクの3/4分の費用を計算

### 2-5. 計算結果の表示
- 合計金額（円）を大きく表示
- 補足情報として以下も表示：
  - 使用タンク容量（L）
  - 燃料種別
  - 適用単価（円/L）

## 3. 画面構成

```
[ヘッダー]
  サイト名 / キャッチコピー

[入力セクション]
  1. 車種を入力・選択
  2. 燃料種別を選択（レギュラー / ハイオク / 軽油）
  3. 現在のガソリン単価（円/L）を入力
  4. 現在の残量（任意：満タン / 3/4 / 1/2 / 1/4 / ほぼ空）

[計算ボタン]

[結果表示セクション]
  合計金額：¥〇〇〇〇
  （タンク容量 〇〇L × 〇〇円/L）

[フッター]
  免責事項・補足
```

## 5. UX・デザイン方針

- **シンプルさ優先**：入力項目を最小限に絞り、3ステップ以内で結果を表示
- **スマートフォン対応**：レスポンシブデザイン。給油所でもサッと使えるUI
- **視認性**：計算結果の金額を画面中央に大きく表示
- **入力補助**：車種はインクリメンタルサーチ（入力途中で候補を表示）
- **カラーテーマ**：ガソリン・エネルギーを連想させるオレンジ・赤系をアクセントに使用

## 6. 追加実装済み機能（2026-04-01）

### 6-1. 全国平均ガソリン価格の表示
- `prices.json` に最新価格（レギュラー・ハイオク・軽油）と更新日を管理
- ページ上部に常時表示（出典：資源エネルギー庁 石油製品価格調査）
- 「使う」ボタンで燃料種別を自動選択し、単価入力欄へ平均価格を自動入力
- 価格更新手順：`prices.json` の値を変更して GitHub へ push → 自動デプロイ

### 6-2. タンク容量の手動入力
- 車種リスト外の車・バイク向けに、タンク容量（L）を直接入力可能
- 車種選択時は自動入力（上書き可能）
- タンク容量のみ入力しても計算可能（車種選択不要）

### 6-3. 前回入力値の保持（localStorage）
- 計算実行時に以下をブラウザに自動保存
  - 車種名 / タンク容量 / 燃料種別 / 単価 / 残量
- 次回アクセス時に自動復元（毎回入力不要）

### 6-4. 計算結果の拡充
- 給油量（L）と単価の内訳を結果に表示
- 「1,000円で約XX L 給油できます」を追加表示
- 車種未選択時はタンク容量を結果に表示

### 6-5. デプロイ自動化
- GitHub Actions（`.github/workflows/deploy.yml`）により、`main` ブランチへの push で Cloudflare Pages へ自動デプロイ
- ホスティング: Cloudflare Pages（`caroilprice.pages.dev`）

## 8. 将来的な拡張アイデア

- **現在地のガソリン価格自動取得**：外部API（gogo.gs 等）連携で最寄りのスタンド価格を取得
- **給油履歴の記録**：ローカルストレージやアカウント登録で給油コストを管理
- **燃費計算**：走行距離÷燃料消費量で燃費（km/L）を記録・表示
- **複数車種の比較**：「この車とあの車、どちらが燃料コストが安い？」の比較機能
- **月間燃料費シミュレーター**：月の走行距離・燃費・単価から月コストを算出

## 10. 免責事項

- タンク容量は公式スペックに基づく参考値です。実際の給油量は車の状態・残量センサーの精度により異なります。
- ガソリン単価は手入力のため、実際の給油所価格と異なる場合があります。
- 本サービスの計算結果はあくまで目安です。
